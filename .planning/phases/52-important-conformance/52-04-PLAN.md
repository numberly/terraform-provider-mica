Resource file (confirmed): `internal/provider/nfs_export_policy_rule_resource.go` — currently at `Schema.Version: 0` with a no-op `UpgradeState` returning an empty map.

Purpose: Enable users to clear the security list through `terraform apply` (setting security = []) — matches convention and closes a real PATCH semantic gap.
Output: Model pointer migration, Update builder update, schema v0→v1 bump, identity upgrader, 2 new tests (clear + upgrader).
