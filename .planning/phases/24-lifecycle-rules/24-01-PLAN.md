API reference — LifecycleRule schema:
```
LifecycleRule (GET response):
  abort_incomplete_multipart_uploads_after (integer)
  bucket (object: NamedReference with name+id)
  cleanup_expired_object_delete_marker (ro boolean)
  context (ro object — ignore, internal FlashBlade field)
  enabled (boolean)
  id (ro string)
  keep_current_version_for (integer)
  keep_current_version_until (integer)
  keep_previous_version_for (integer)
  name (ro string — composite "bucketName/ruleId")
  prefix (string)
  rule_id (string)

LifecycleRulePost (POST body):
  abort_incomplete_multipart_uploads_after (integer)
  bucket (object: {name: string})
  keep_current_version_for (integer)
  keep_current_version_until (integer)
  keep_previous_version_for (integer)
  prefix (string)
  rule_id (string)

LifecycleRulePatch (PATCH body):
  abort_incomplete_multipart_uploads_after (integer)
  enabled (boolean)
  keep_current_version_for (integer)
  keep_current_version_until (integer)
  keep_previous_version_for (integer)
  prefix (string)

API endpoints:
  GET    /api/2.22/lifecycle-rules?bucket_ids=&bucket_names=
  POST   /api/2.22/lifecycle-rules?confirm_date=   (body: LifecycleRulePost)
  PATCH  /api/2.22/lifecycle-rules?bucket_ids=&bucket_names=&confirm_date=  (body: LifecycleRulePatch)
  DELETE /api/2.22/lifecycle-rules?bucket_ids=&bucket_names=
```
