- `RemoteCredentialsName` types.String (Optional — for S3 replication targets; omit for FB-to-FB)
- `RemoteName` types.String (Computed — populated from API response, the array connection remote)
- `Paused` types.Bool (Optional, Computed, Default false)
- `CascadingEnabled` types.Bool (Optional, Computed, Default false, RequiresReplace — immutable after creation)
- `Direction` types.String (Computed)
- `Status` types.String (Computed)
- `StatusDetails` types.String (Computed)
- `Lag` types.Int64 (Computed, int64UseStateForUnknown)
- `RecoveryPoint` types.Int64 (Computed, int64UseStateForUnknown)
- `ObjectBacklogCount` types.Int64 (Computed, int64UseStateForUnknown — flattened from object_backlog.count)
- `ObjectBacklogTotalSize` types.Int64 (Computed, int64UseStateForUnknown — flattened from object_backlog.total_size)
- `Timeouts` timeouts.Value

Resource struct: `bucketReplicaLinkResource` with `client *client.FlashBladeClient`.
Type name: `flashblade_bucket_replica_link`.
Implement: Resource, ResourceWithConfigure, ResourceWithImportState, ResourceWithUpgradeState.
SchemaVersion: 0, UpgradeState returns empty map.

**Create**: Call `r.client.PostBucketReplicaLink(ctx, localBucketName, remoteBucketName, remoteCredentialsName, body)`. Map response to model.

**Read**: Call `r.client.GetBucketReplicaLink(ctx, localBucketName, remoteBucketName)`. Map response to model. Handle IsNotFound by removing from state.

**Update**: Only `paused` is mutable via PATCH. Build BucketReplicaLinkPatch if paused changed. Call PatchBucketReplicaLink using the resource ID. Then re-read for computed fields.

**Delete**: Call `r.client.DeleteBucketReplicaLink(ctx, localBucketName, remoteBucketName)`. Handle IsNotFound gracefully.

**ImportState**: Import by composite ID `localBucket/remoteBucket` using `parseCompositeID(id, 2)`. Call GetBucketReplicaLink with the two parts. Map to model. Set Timeouts to null object.

**Mapping function** `mapBucketReplicaLinkToModel(link *client.BucketReplicaLink, data *bucketReplicaLinkModel)`:
- data.ID = link.ID
- data.LocalBucketName = link.LocalBucket.Name
- data.RemoteBucketName = link.RemoteBucket.Name
- data.RemoteName = stringOrNull(link.Remote.Name)
- data.RemoteCredentialsName: if link.RemoteCredentials != nil, set to link.RemoteCredentials.Name; else preserve existing state value (Optional field)
- data.Paused = link.Paused
- data.CascadingEnabled = link.CascadingEnabled
- data.Direction = stringOrNull(link.Direction)
- data.Status = stringOrNull(link.Status)
- data.StatusDetails = stringOrNull(link.StatusDetails)
- data.Lag = types.Int64Value(link.Lag)
- data.RecoveryPoint = types.Int64Value(link.RecoveryPoint)
- data.ObjectBacklogCount and data.ObjectBacklogTotalSize: if link.ObjectBacklog != nil, set from it; else types.Int64Value(0)
