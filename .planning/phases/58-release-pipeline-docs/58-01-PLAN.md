## Critical decisions from STATE.md

- Plugin binary name: `pulumi-resource-flashblade` (Pulumi CLI convention)
- Archive name template: `pulumi-resource-flashblade-v{VERSION}-{os}-{arch}.tar.gz` (Pulumi plugin install expects this format)
- PluginDownloadURL: `github://api.github.com/numberly` (already in resources.go)
- Go SDK tag pattern: `sdk/go/vX.Y.Z` (required for `go get` with GOPRIVATE)
- Cosign: keyless Sigstore (reuse existing `.goreleaser.yml` pattern exactly)
- Version source: `git describe --tags --dirty --always` (Makefile pattern)
- Schema artifacts: `schema-embed.json` + `bridge-metadata.json` embedded in binary via `//go:embed`

## Pulumi-specific goreleaser requirements

Unlike the TF provider, the Pulumi plugin:
1. Binary name is `pulumi-resource-flashblade` (no `_v{VERSION}` suffix — Pulumi CLI resolves by folder name inside archive)
2. Archive format is `.tar.gz` (not `.zip` — Pulumi CLI extracts tar.gz)
3. Archive must contain the binary at the root (not in a subfolder)
4. No `archives.formats: [zip]` — use `tar.gz` only
5. `project_name` should be `pulumi-resource-flashblade` for checksum naming

Create `.github/workflows/pulumi-release.yml` with the following exact structure:

```yaml
---
name: Pulumi Release

on:
  push:
    tags:
      - "pulumi-*"

permissions:
  contents: write
  id-token: write
  attestations: write

jobs:
  prerequisites:
    name: Prerequisites
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Setup Go
        uses: actions/setup-go@v5
        with:
          go-version-file: pulumi/provider/go.mod
          cache: true

      - name: Run provider unit tests
        working-directory: pulumi
        run: make test

      - name: Generate schema artifacts
        working-directory: pulumi
        run: make tfgen

      - name: Schema drift gate
        run: |
          git diff --exit-code \
            pulumi/provider/cmd/pulumi-resource-flashblade/schema.json \
            pulumi/provider/cmd/pulumi-resource-flashblade/schema-embed.json \
            pulumi/provider/cmd/pulumi-resource-flashblade/bridge-metadata.json

      - name: Upload schema-embed.json artifact
        uses: actions/upload-artifact@v4
        with:
          name: schema-embed
          path: pulumi/provider/cmd/pulumi-resource-flashblade/schema-embed.json

      - name: Upload bridge-metadata.json artifact
        uses: actions/upload-artifact@v4
        with:
          name: bridge-metadata
          path: pulumi/provider/cmd/pulumi-resource-flashblade/bridge-metadata.json

  release_provider:
    name: Release Provider
    needs: prerequisites
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Setup Go
        uses: actions/setup-go@v5
        with:
          go-version-file: pulumi/provider/go.mod
          cache: true

      - name: Download schema-embed.json artifact
        uses: actions/download-artifact@v4
        with:
          name: schema-embed
          path: pulumi/provider/cmd/pulumi-resource-flashblade/

      - name: Download bridge-metadata.json artifact
        uses: actions/download-artifact@v4
        with:
          name: bridge-metadata
          path: pulumi/provider/cmd/pulumi-resource-flashblade/

      - name: Install cosign
        uses: sigstore/cosign-installer@v3

      - name: Run GoReleaser
        uses: goreleaser/goreleaser-action@v6
        with:
          version: "~> v2"
          args: release --clean --config pulumi/.goreleaser.pulumi.yml
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

  release_python_sdk:
    name: Release Python SDK
    needs: prerequisites
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup Go
        uses: actions/setup-go@v5
        with:
          go-version-file: pulumi/provider/go.mod
          cache: true

      - name: Setup Python
        uses: actions/setup-python@v5
        with:
          python-version: "3.12"

      - name: Download schema-embed.json artifact
        uses: actions/download-artifact@v4
        with:
          name: schema-embed
          path: pulumi/provider/cmd/pulumi-resource-flashblade/

      - name: Generate Python SDK
        working-directory: pulumi
        run: make generate_python

      - name: Build Python wheel
        working-directory: pulumi
        run: make test_python_sdk

      - name: Upload wheel to release
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: |
          VERSION="${GITHUB_REF_NAME#pulumi-}"
          WHEEL_FILE=$(ls pulumi/sdk/python/dist/pulumi_flashblade-*.whl | head -n1)
          if [ -z "$WHEEL_FILE" ]; then
            echo "Wheel file not found"
            exit 1
          fi
          gh release upload "${GITHUB_REF_NAME}" "$WHEEL_FILE" --clobber

  tag_go_sdk:
    name: Tag Go SDK
    needs: release_provider
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Push Go SDK tag
        run: |
          VERSION="${GITHUB_REF_NAME#pulumi-}"
          GO_TAG="sdk/go/v${VERSION}"
          git tag "${GO_TAG}" "${GITHUB_SHA}"
          git push origin "${GO_TAG}"
```

Key design decisions:
- Trigger: `tags: ["pulumi-*"]` (not `v*` — isolates from TF provider tags)
- `fetch-depth: 0` required for GoReleaser changelog generation
- Permissions: `contents: write` (release upload), `id-token: write` (cosign OIDC), `attestations: write` (GitHub attestations)
- 4 jobs in dependency chain: prerequisites → (release_provider + release_python_sdk in parallel) → tag_go_sdk
- `release_provider` downloads both schema-embed.json and bridge-metadata.json artifacts (both needed for `//go:embed`)
- `tag_go_sdk` strips `pulumi-` prefix from tag to create `sdk/go/vX.Y.Z` (e.g., tag `pulumi-2.22.3` → `sdk/go/v2.22.3`)
- Wheel upload uses `gh release upload` CLI (available in `ubuntu-latest` runner) with `--clobber` for idempotency
- `GORELEASER_CURRENT_TAG` env var not needed — goreleaser-action auto-detects from `GITHUB_REF_NAME`
