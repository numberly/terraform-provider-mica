type certificateGroupModel struct {
    ID       types.String   `tfsdk:"id"`

Schema (Version: 0):
- `id`: Computed, UseStateForUnknown (stable after creation)
- `name`: Required, RequiresReplace (immutable)
- `realms`: Computed, `ElementType: types.StringType` — NO UseStateForUnknown (volatile, must detect drift)
- `timeouts`: Create(20m), Read(5m), Delete(30m) — NO Update (CRD resource)

**No PATCH endpoint exists** — resource is CRD (Create/Read/Delete only):
- Implement `Create`, `Read`, `Delete` normally
- `Update` method must AddError: "certificate groups cannot be updated — all attributes force a new resource"
- `UpgradeState` returns `map[int64]resource.StateUpgrader{}` (Version 0, no prior states)

**Create**:
```go
group, err := r.client.PostCertificateGroup(ctx, data.Name.ValueString())
```
Map response: ID via `types.StringValue(group.ID)`, Name via `types.StringValue(group.Name)`, Realms via listToStringSlice helper or inline `types.ListValueMust(types.StringType, ...)`.

**Read** (drift detection mandatory):
```go
group, err := r.client.GetCertificateGroup(ctx, data.Name.ValueString())
```
For each field — log drift if changed:
```go
if data.Name.ValueString() != group.Name {
    tflog.Debug(ctx, "drift detected", map[string]any{
        "resource": data.Name.ValueString(), "field": "name",
        "was": data.Name.ValueString(), "now": group.Name,
    })
}
// Same for realms (compare stringified or element-by-element)
```
Map realms using the empty-safe pattern (NOT types.ListNull — avoids perpetual drift):
```go
if len(group.Realms) == 0 {
    data.Realms = types.ListValueMust(types.StringType, []attr.Value{})
} else {
    elems := make([]attr.Value, len(group.Realms))
    for i, r := range group.Realms {
        elems[i] = types.StringValue(r)
    }
    data.Realms = types.ListValueMust(types.StringType, elems)
}
```

**Delete**: call `r.client.DeleteCertificateGroup`, handle IsNotFound as no-op.

**ImportState**:
```go
func (r *certificateGroupResource) ImportState(ctx context.Context, req resource.ImportStateRequest, resp *resource.ImportStateResponse) {
    name := req.ID
    group, err := r.client.GetCertificateGroup(ctx, name)
    // handle error...
    var data certificateGroupModel
    data.Timeouts = nullTimeoutsValue()  // CRD: create/read/delete only
    data.ID = types.StringValue(group.ID)
    data.Name = types.StringValue(group.Name)

    resp.Diagnostics.Append(resp.State.Set(ctx, &data)...)
}
```
