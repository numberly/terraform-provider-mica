<objective>
Fix OAuth2 context propagation and remove the RetryBaseDelay duration heuristic from the client package.

Purpose: CH-01 ensures that when Terraform cancels an operation (e.g. SIGINT), the in-flight OAuth2 token exchange is also cancelled rather than blocking indefinitely on context.Background(). CH-02 removes a footgun where callers could accidentally pass raw integer values (RetryBaseDelay: 1 meaning 1ms) instead of proper time.Duration values.

Output:
- auth.go: FlashBladeTokenSource stores a context passed at construction time; Token() uses it
- client.go: RetryBaseDelay field removed from Config; default 1000ms applied inline; heuristic branch deleted
- All callers (provider.go, *_test.go) updated to compile cleanly
</objective>
