- ID (types.String), Name (types.String), Address (types.String), SubnetName (types.String), Type (types.String), Services (types.String — single value, NOT list), AttachedServers (types.List of StringType), Enabled (types.Bool), Gateway (types.String), MTU (types.Int64), Netmask (types.String), VLAN (types.Int64), Realms (types.List of StringType), Timeouts (timeouts.Value)

**Schema:** `flashblade_network_interface`, Version 0. Attributes:
- `id`: Computed, UseStateForUnknown
- `name`: Required, RequiresReplace (user-provided name, per user decision)
- `subnet_name`: Required, RequiresReplace (immutable after creation)
- `type`: Required, RequiresReplace (immutable, only valid value is "vip")
- `services`: Required, StringAttribute with enum validator accepting: data, sts, egress-only, replication
- `address`: Required, StringAttribute
- `attached_servers`: Optional+Computed, ListAttribute of StringType. Description: "Required for data/sts; forbidden for egress-only/replication."
- `enabled`: Computed, Bool, UseStateForUnknown
- `gateway`: Computed, String, UseStateForUnknown
- `mtu`: Computed, Int64, UseStateForUnknown
- `netmask`: Computed, String, UseStateForUnknown
- `vlan`: Computed, Int64, UseStateForUnknown
- `realms`: Computed, List of StringType, UseStateForUnknown
- `timeouts`: timeouts.Attributes with Create, Read, Update, Delete

**Validator — serviceTypeValidator():** Create a `validator.String` that accepts exactly: "data", "sts", "egress-only", "replication". Use the established pattern from validators.go (check `stringOneOfValidator` or similar if it exists, otherwise create inline).

**ConfigValidators — services x attached_servers cross-field validation (NI-06):**
```go
var _ resource.ResourceWithConfigValidators = &networkInterfaceResource{}

func (r *networkInterfaceResource) ConfigValidators(_ context.Context) []resource.ConfigValidator {
    return []resource.ConfigValidator{networkInterfaceServicesValidator{}}
}
```
The `networkInterfaceServicesValidator` must:
- Return early if Services or AttachedServers is Unknown (defer to apply)
- For services "data" or "sts": require attached_servers with exactly 1 element
- For services "egress-only" or "replication": require attached_servers to be null or empty

**Create:** Build `client.NetworkInterfacePost` with Address, Services (wrap single string into []string), Subnet (&NamedReference{Name: subnetName}), Type, AttachedServers (expand list to []NamedReference using helper). Call `PostNetworkInterface(ctx, name, body)`. Map response via `mapNetworkInterfaceToModel`.

**Read:** Get by name. If NotFound, remove resource. Log drift on address, services, attached_servers changes. Map via `mapNetworkInterfaceToModel`.

**Update:** Compare plan vs state. Build `client.NetworkInterfacePatch`:
- Address: set *string pointer only if changed
- Services: always include ([]string with single value from plan) — needed because field has no omitempty
- AttachedServers: always include (full-replace semantics) — expand from plan list to []NamedReference
Call `PatchNetworkInterface(ctx, name, patch)`. Map response.

IMPORTANT for Update: The PATCH struct has Services and AttachedServers without omitempty, so they will ALWAYS be sent. This is correct — full-replace semantics. Only set Address pointer when it actually changed to avoid unnecessary writes.

**Delete:** Call `DeleteNetworkInterface`. Ignore NotFound.

**ImportState:** Set Name from req.ID, set Timeouts to nullTimeoutsValue(), call GetNetworkInterface, map to model.

**Helper — `mapNetworkInterfaceToModel`:** Maps `*client.NetworkInterface` to `*networkInterfaceResourceModel`:
- ID, Name, Address, Type, Enabled, MTU, VLAN as direct value mappings
- Gateway, Netmask: use `stringOrNull` helper
- SubnetName: extract from `ni.Subnet` — if nil or Name empty, types.StringNull()
- Services: if len > 0, `types.StringValue(ni.Services[0])`, else `types.StringNull()` (collapse []string to single string)
- AttachedServers: if len > 0, build []string from []NamedReference names, then `types.ListValueFrom(ctx, types.StringType, names)`. If len == 0, use `types.ListValueFrom(ctx, types.StringType, []string{})` — empty list, NOT null, to avoid spurious drift.
- Realms: if len > 0, `types.ListValueFrom`, else `types.ListNull(types.StringType)`

**Helper — `niServersToNamedRefs`:** Converts types.List to []client.NamedReference. If list is null/unknown/empty, return nil.

**Helper — `refToSubnetName`:** Converts *client.NamedReference to types.String (same pattern as refToLagName).
