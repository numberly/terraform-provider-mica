**In `internal/client/models_network.go`**, append three structs after the existing `LinkAggregationGroup` struct:

1. `NetworkInterface` (GET response) — fields: ID (string, omitempty), Name (string), Address (string, omitempty), Enabled (bool, omitempty), Gateway (string, omitempty), MTU (int64, omitempty), Netmask (string, omitempty), Services ([]string, omitempty), Subnet (*NamedReference, omitempty), Type (string, omitempty), VLAN (int64, omitempty), AttachedServers ([]NamedReference, json:"attached_servers,omitempty"), Realms ([]string, json:"realms,omitempty").

2. `NetworkInterfacePost` (POST body) — writable fields only: Address (string, omitempty), Services ([]string, omitempty), Subnet (*NamedReference, omitempty), Type (string, omitempty), AttachedServers ([]NamedReference, json:"attached_servers,omitempty"). Name is NOT in the body — passed via ?names= query param.

3. `NetworkInterfacePatch` (PATCH body) — mutable fields only: Address (*string, json:"address,omitempty"), Services ([]string, json:"services"), AttachedServers ([]NamedReference, json:"attached_servers"). CRITICAL: Services and AttachedServers must NOT have `omitempty` — clearing them requires sending `[]` in JSON. Address uses *string + omitempty for true PATCH semantics (only included when changed).

- `GetNetworkInterface(ctx, name)` — uses `getOneByName[NetworkInterface]` with path `/network-interfaces?names=`
- `ListNetworkInterfaces(ctx)` — GET `/network-interfaces`, returns `[]NetworkInterface`
- `PostNetworkInterface(ctx, name, body NetworkInterfacePost)` — POST `/network-interfaces?names=` + url.QueryEscape(name)
- `PatchNetworkInterface(ctx, name, body NetworkInterfacePatch)` — PATCH `/network-interfaces?names=` + url.QueryEscape(name)
- `DeleteNetworkInterface(ctx, name)` — DELETE `/network-interfaces?names=` + url.QueryEscape(name)

Import `context`, `fmt`, `net/url`. Follow `subnets.go` exactly for error handling and ListResponse unwrapping.

  <name>Task 2: Create mock handler for network interfaces</name>
  <files>internal/testmock/handlers/network_interfaces.go</files>
  <action>
Create `internal/testmock/handlers/network_interfaces.go` following `subnets.go` pattern exactly. Key differences:

**Store struct** `networkInterfaceStore` with `sync.Mutex`, `byName map[string]*client.NetworkInterface`, `byID map[string]*client.NetworkInterface`.

**`RegisterNetworkInterfaceHandlers(mux *http.ServeMux) *networkInterfaceStore`** — registers on `/api/2.22/network-interfaces`.

**`AddNetworkInterface(name, address, subnetName, niType, service string) *client.NetworkInterface`** — seeder method for tests. Creates a NI with:
- UUID id, Enabled=true, the provided name/address/type
- Subnet = &NamedReference{Name: subnetName}
- Services = []string{service}
- AttachedServers = nil (empty by default; tests that need servers call a separate method or set directly)
- Set Gateway, MTU, Netmask, VLAN to sensible defaults (e.g., Gateway="10.21.200.1", MTU=1500, Netmask="255.255.255.0", VLAN=0)

**handlePost** — reads name from `?names=` query param (error 400 if missing). Decodes `client.NetworkInterfacePost` from body. Checks conflict. Creates `client.NetworkInterface` merging name from query param + body fields. Derives computed fields: Enabled=true, Gateway/MTU/Netmask/VLAN from sensible defaults (in real API these come from subnet, mock just uses defaults).

**handlePatch** — reads name from `?names=`. Uses raw `map[string]json.RawMessage` decoding (same as subnet mock). For `address`: unmarshal string, update. For `services`: unmarshal `[]string`, **full-replace** (overwrite entire slice). For `attached_servers`: unmarshal `[]client.NamedReference`, **full-replace** (overwrite entire slice). Reject any other fields silently (don't error, just ignore — consistent with API behavior).

**handleGet** — same as subnet: optional `?names=` filter, returns matching or all.

**handleDelete** — same as subnet: `?names=` required, 404 if not found, remove from both maps.

Import: `encoding/json`, `fmt`, `net/http`, `sync`, `github.com/google/uuid`, client package.
