type QosPolicy struct {
	ID                  string `json:"id"`
	Name                string `json:"name"`
	Enabled             bool   `json:"enabled"`

	MaxTotalOpsPerSec   int64  `json:"max_total_ops_per_sec"`

	Name                string `json:"name"`

	Name                *string `json:"name,omitempty"`
}

1. Create `internal/provider/qos_policy_resource.go` following lifecycle_rule_resource.go pattern:

   **Struct:** `qosPolicyResource` with `client *client.FlashBladeClient`
   **Interfaces:** Resource, ResourceWithConfigure, ResourceWithImportState
   **Factory:** `NewQosPolicyResource()`
   **TypeName:** `flashblade_qos_policy`

   **Model:** `qosPolicyModel`
   - ID types.String (Computed, UseStateForUnknown)
   - Name types.String (Required, RequiresReplace — API does not support name change in practice)
   - Enabled types.Bool (Optional, Computed, Default: true)
   - MaxTotalBytesPerSec types.Int64 (Optional — bandwidth limit)
   - MaxTotalOpsPerSec types.Int64 (Optional — IOPS limit)
   - IsLocal types.Bool (Computed — read-only from API)
   - PolicyType types.String (Computed — read-only from API, will be "bandwidth-limit")
   - Timeouts timeouts.Value (Create, Read, Update, Delete)

   **Create:** Extract plan, build QosPolicyPost{Name, Enabled (pointer), MaxTotalBytesPerSec, MaxTotalOpsPerSec}. Only include byte/ops fields if not null/unknown (same IsNull/IsUnknown check as lifecycle rules). Call PostQosPolicy.

   **Read:** Get state, call GetQosPolicy(ctx, name). If IsNotFound, RemoveResource. Map response.

   **Update:** Compare plan vs state. Build QosPolicyPatch with pointer fields for changed values only. Call PatchQosPolicy if any changes. Re-read to refresh computed fields.

   **Delete:** Call DeleteQosPolicy(ctx, name). Handle IsNotFound gracefully.

   **ImportState:** Import ID is the policy name (single string). Call GetQosPolicy, map to model with nullTimeoutsValue().

   **Helper:** `mapQosPolicyToModel(policy *client.QosPolicy, data *qosPolicyModel)` — maps API response. For MaxTotalBytesPerSec and MaxTotalOpsPerSec: if API returns 0, map to types.Int64Null() if model value is null/unknown (same pattern as lifecycle rule optional int64 fields).
