    Create internal/provider/object_store_user_resource.go implementing the flashblade_object_store_user resource.

    Resource model struct objectStoreUserModel:
    - ID types.String `tfsdk:"id"` — Computed, UseStateForUnknown
    - Name types.String `tfsdk:"name"` — Required, RequiresReplace
    - FullAccess types.Bool `tfsdk:"full_access"` — Optional, Computed (defaults to false if not set; read from API on refresh)
    - Timeouts timeouts.Value `tfsdk:"timeouts"`

    Schema version 0, UpgradeState returns map[int64]resource.StateUpgrader{} (new resource).

    Implement resource interfaces:
    - resource.Resource
    - resource.ResourceWithConfigure
    - resource.ResourceWithImportState
    - resource.ResourceWithUpgradeState

    Create method:
    1. Get plan into model
    2. Apply create timeout (20 min default)
    3. Build ObjectStoreUserPost{FullAccess: ptr(data.FullAccess.ValueBool())} only if FullAccess is explicitly set (not null/unknown); send nil body otherwise
    4. Call PostObjectStoreUser(ctx, data.Name.ValueString(), body)
    5. Map API response to model (id, name, full_access)
    6. Set state

    Read method:
    1. Get state into model
    2. Apply read timeout (5 min default)
    3. Call GetObjectStoreUser(ctx, name) — if IsNotFound, RemoveResource
    4. Drift detection: if resp.FullAccess != data.FullAccess.ValueBool(), log with tflog.Warn including resource name, field "full_access", old value, new value
    5. Map API response to model
    6. Set state

    Delete method:
    1. Get state into model
    2. Apply delete timeout (30 min default)
    3. Call DeleteObjectStoreUser — if IsNotFound, return (idempotent)
    4. Errors surface as diagnostic

    ImportState method:
    1. Import ID is the user name (format: account/username) — use req.ID directly
    2. Call GetObjectStoreUser to verify existence
    3. Populate model and set state
    4. Timeouts initialized to null object (same pattern as qos_policy_member_resource.go)

    Update is not supported — add stub returning AddError "Update not supported".

    Helper mapObjectStoreUserToModel(user *client.ObjectStoreUser, data *objectStoreUserModel) sets id, name, full_access.
