# Phase 56, Plan 02: Python SDK Generation and Wheel Build

## Objective

Generate a working Python SDK under `pulumi/sdk/python/` with `pulumi_flashblade` import path. Verify `python -m build` produces an installable `.whl` file.

## Context

Python SDK generation for Pulumi bridged providers typically uses `pulumi-tfgen` with a language-specific subcommand or the `pulumi` CLI's `gen-sdk` command. The generated SDK must:
- Use package name `pulumi_flashblade`
- Be buildable with `python -m build` (PEP 517)
- Produce a `.whl` installable via `pip install`

The existing `pulumi/sdk/python/` directory does not exist yet. The `schema.json` contains a `language.python` block with `pyproject: {}`.

Add a `test_python_sdk` target to `pulumi/Makefile` that verifies the Python SDK builds correctly:

```makefile
	cd $(PYTHON_SDK_DIR) && python -m build

	@echo "Python SDK build verified"
```
