1. Define `FlashBladeProvider` struct with `version` string field.
2. Define `New(version string) func() provider.Provider` factory function.
3. Implement `provider.Provider` interface:

**Schema method** — per CONTEXT.md locked decisions:
```hcl
provider "flashblade" {
  endpoint            = "..."     # Optional, falls back to FLASHBLADE_HOST
  ca_cert_file        = "..."     # Optional, path to CA cert PEM
  ca_cert             = "..."     # Optional, inline CA cert PEM
  insecure_skip_verify = false    # Optional, default false
  max_retries         = 3         # Optional, default 3
  retry_base_delay    = "1s"      # Optional, default "1s"
  auth {                          # Required block
    api_token = "..."             # Optional (Sensitive), falls back to FLASHBLADE_API_TOKEN
    oauth2 {                      # Optional sub-block
      client_id = "..."           # Optional (Sensitive), falls back to FLASHBLADE_OAUTH2_CLIENT_ID
      key_id    = "..."           # Optional (Sensitive), falls back to FLASHBLADE_OAUTH2_KEY_ID
      issuer    = "..."           # Optional, falls back to FLASHBLADE_OAUTH2_ISSUER
    }
  }
}
```

Use `schema.SingleNestedAttribute` for `auth` block. Use `schema.SingleNestedAttribute` for `oauth2` sub-block. Mark `api_token`, `client_id`, `key_id` as `Sensitive: true` (PROV-05).

4. Define Go model structs matching schema:
   - `flashBladeProviderModel` with fields for each attribute
   - `authModel` with APIToken and OAuth2 sub-model
   - `oauth2Model` with ClientID, KeyID, Issuer

5. **Configure method:**
   a. Parse config into model struct.
   b. Resolve env var fallbacks (PROV-03):
      - endpoint: config.Endpoint.ValueString() -> os.Getenv("FLASHBLADE_HOST")
      - api_token: config.Auth.APIToken.ValueString() -> os.Getenv("FLASHBLADE_API_TOKEN")
      - oauth2 fields: similar pattern with FLASHBLADE_OAUTH2_* prefix
   c. Validate: endpoint must be set (from config or env). Either api_token or full oauth2 block must be set.
   d. Log configuration with tflog.Info: endpoint (masked), auth mode (token vs oauth2) (PROV-06).
   e. Build `client.Config` from resolved values, using sensible defaults (MaxRetries=3, RetryBaseDelay=1s).
   f. Parse `retry_base_delay` string as `time.Duration`.
   g. Call `client.NewClient(cfg)` — on error, add diagnostic.
   h. Call `client.NegotiateVersion(ctx)` — on error, add diagnostic (PROV-04).
   i. Set `resp.ResourceData = c` and `resp.DataSourceData = c`.

6. **Metadata method:** Set TypeName to `flashblade`.

7. **Resources method:** Return empty slice (file system resource added in Plan 04).

8. **DataSources method:** Return empty slice (file system data source added in Plan 04).
