   **Factory:** `NewBucketAuditFilterResource()`
   **TypeName:** `flashblade_bucket_audit_filter`

   **Model:** `bucketAuditFilterModel`
   - BucketName types.String (Required, RequiresReplace â€” one filter per bucket, bucket is the key)
   - Actions types.List of types.StringType (Required â€” list of audit action strings)
   - S3Prefixes types.List of types.StringType (Optional, Computed â€” list of S3 prefix strings, defaults to empty list)
   - Timeouts timeouts.Value (Create, Read, Update, Delete)

   Note: No `id` attribute needed as Terraform state key. The bucket_name IS the identifier (one audit filter per bucket, like bucket access policies). But include `id` as Computed for consistency with other resources if the API returns a name field.

   **Schema:** bucket_name (Required, string, RequiresReplace), actions (Required, list of string), s3_prefixes (Optional, Computed, list of string), timeouts (CRUD).

   **Create:** Extract plan, build BucketAuditFilterPost{Actions, S3Prefixes}, call PostBucketAuditFilter(ctx, bucketName, body), map response to model.

   **Read:** Get state, call GetBucketAuditFilter(ctx, bucketName). If IsNotFound, RemoveResource. Map response to model.

   **Update:** Compare plan vs state for actions and s3_prefixes. Build BucketAuditFilterPatch with pointer fields only for changed values. Call PatchBucketAuditFilter. Re-read to refresh.

   **Delete:** Call DeleteBucketAuditFilter(ctx, bucketName). Handle IsNotFound gracefully.

   **ImportState:** Import ID is the bucket name (single string, no composite). Call GetBucketAuditFilter, map to model with nullTimeoutsValue().

   **Helper:** `mapBucketAuditFilterToModel(filter *client.BucketAuditFilter, data *bucketAuditFilterModel)` â€” maps API response to Terraform model. Convert []string to types.ListValueFrom for actions and s3_prefixes. Use types.ListValueMust with types.StringType for the conversion.
