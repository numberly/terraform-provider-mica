lifecycle_rule resource: bucket_name (Required), rule_id (Computed), prefix, enabled, abort_incomplete_multipart_uploads_after (Int64), keep_current_version_for (Int64), keep_current_version_until (Int64), keep_previous_version_for (Int64), cleanup_expired_object_delete_marker (Bool)

bucket_access_policy resource: bucket_name (Required), enabled (Computed)

bucket_access_policy_rule resource: name (Required), bucket_name (Required), actions (List), effect, principals (List), resources (List)

bucket_audit_filter resource: bucket_name (Required), actions (List), s3_prefixes (List)

qos_policy resource: name (Required), enabled (Bool), max_total_bytes_per_sec (Int64), max_total_ops_per_sec (Int64), is_local (Computed), policy_type (Computed)

qos_policy_member resource: policy_name (Required), member_name (Required), member_type (Required)

<!-- Data source schemas -->

lifecycle_rule DS: bucket_name (Required), rule_id (Required), + all rule attrs (Computed)
bucket_access_policy DS: bucket_name (Required), id/enabled/rule_count (Computed)
bucket_audit_filter DS: bucket_name (Required), actions/s3_prefixes (Computed)
qos_policy DS: name (Required), id/enabled/max_total_bytes_per_sec/max_total_ops_per_sec/is_local/policy_type (Computed)

<!-- Mock handler registration -->

handlers.RegisterBucketAccessPolicyHandlers(ms.Mux) returns *bucketAccessPolicyStore
store.Seed(&client.BucketAccessPolicy{...}) seeds a policy

client.BucketAccessPolicy: ID, Bucket (NamedReference), Enabled (bool), Rules ([]BucketAccessPolicyRule)
client.NamedReference: Name string
