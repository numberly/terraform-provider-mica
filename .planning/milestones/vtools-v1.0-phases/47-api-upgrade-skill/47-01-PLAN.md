- In client.go: exact regex `const APIVersion = "OLD"` → `const APIVersion = "NEW"`
- In server.go: exact regex `"OLD"` only in the versions slice line (line containing `"versions":`)
- In handlers/*.go: simple string replace `/api/OLD/` → `/api/NEW/`

  <action>
Create `.claude/skills/api-upgrade/scripts/upgrade_version.py` using Python 3.10+ stdlib only (re, pathlib, argparse, sys).

**CLI interface:**
```
python3 upgrade_version.py \
  --from 2.22 \
  --to 2.23 \
  [--project-root /path/to/provider] \
  [--dry-run | --apply]
```
`--dry-run` is the default (add `default=True` / mutual exclusion via `add_mutually_exclusive_group`).

**Three replacement strategies — implement as separate functions:**

1. `replace_client_version(content, old, new) -> (new_content, list[Change])`:
   - Regex: `r'(const APIVersion = )"` + re.escape(old) + r'"'`
   - Replace with group(1) + `"` + new + `"`
   - Returns list of `Change(lineno, old_line, new_line)` namedtuples

2. `replace_server_versions(content, old, new) -> (new_content, list[Change])`:
   - Only replace `"OLD"` on the line containing `"versions":` (find that line first, then replace within it)
   - Must NOT replace older version strings on the same line

3. `replace_handler_paths(content, old, new) -> (new_content, list[Change])`:
   - Simple str.replace(`/api/{old}/`, `/api/{new}/`) per line
   - Collect all changed lines with their line numbers

**`find_replacements(project_root, old, new) -> list[FileChange]`:**
```python
FileChange = namedtuple("FileChange", ["path", "changes"])
```
- Runs all three strategies against the three target sets
- Returns list; caller decides dry-run vs apply

**`apply_replacements(file_changes)`:**
- For each FileChange: `path.write_text(new_content, encoding="utf-8")`

**`main()`:**
- Parse args; default --dry-run=True
- Call find_replacements()
- If no changes found: print warning "No replacements found for version OLD", exit 1
- Print report (dry-run header or apply header)
- If --apply: call apply_replacements(), print "Done. N files modified."

**Error handling:**
- `client.go` or `server.go` not found → print warning, skip (don't exit 1 — allow partial runs)
- handlers dir not found → print warning, skip

Make the script executable: add `if __name__ == "__main__": main()` at the bottom.
