```go
    IsLocal                           types.Bool   `tfsdk:"is_local"`
    MinTlsVersion                     types.String `tfsdk:"min_tls_version"`
    PolicyType                        types.String `tfsdk:"policy_type"`
    TrustedClientCertificateAuthority types.String `tfsdk:"trusted_client_certificate_authority"`
    VerifyClientCertificateTrust      types.Bool   `tfsdk:"verify_client_certificate_trust"`
    Timeouts                          timeouts.Value `tfsdk:"timeouts"`
}
```

Schema (Version: 0):
- `id`: Computed, UseStateForUnknown (stable)
- `name`: Required, RequiresReplace
- `appliance_certificate`: Optional — flat string (certificate name), not nested object. Maps to NamedReference.Name.
- `client_certificates_required`: Optional, Computed (API defaults)
- `disabled_tls_ciphers`: Optional, List of strings
- `enabled`: Optional, Computed (API defaults to true)
- `enabled_tls_ciphers`: Optional, List of strings
- `is_local`: Computed, NO UseStateForUnknown (can change)
- `min_tls_version`: Optional, Computed
- `policy_type`: Computed, NO UseStateForUnknown (can change)
- `trusted_client_certificate_authority`: Optional — flat string (cert/group name)
- `verify_client_certificate_trust`: Optional, Computed
- Timeouts: Create, Read, Update, Delete

Create: Build TlsPolicyPost from plan. For appliance_certificate/trusted_client_certificate_authority, if not null, wrap in `&NamedReference{Name: val}`. Handle cipher lists: convert types.List to []string.

Read: GET by name, drift detection on ALL mutable/computed fields (appliance_certificate, client_certificates_required, disabled_tls_ciphers, enabled, enabled_tls_ciphers, is_local, min_tls_version, policy_type, trusted_client_certificate_authority, verify_client_certificate_trust). Use tflog.Debug with resource/field/was/now keys. For NamedReference fields, compare .Name string. For list fields, compare joined strings.

Update: Build TlsPolicyPatch comparing plan vs state for each field. Only set non-nil pointers for changed fields. For ref fields, use **NamedReference pattern from CONVENTIONS.md.

Delete: Standard delete, handle IsNotFound gracefully.

ImportState: GET by name, nullTimeoutsValue(), mapTlsPolicyToModel.

UpgradeState: empty map (version 0).

`mapTlsPolicyToModel(obj *client.TlsPolicy, data *tlsPolicyModel)` helper:
- For NamedReference fields: if obj.ApplianceCertificate != nil, set types.StringValue(obj.ApplianceCertificate.Name), else types.StringNull()
- For []string cipher lists: if nil/empty set types.ListValueMust(types.StringType, []attr.Value{}), else convert each element
- For booleans: types.BoolValue(obj.Field)
