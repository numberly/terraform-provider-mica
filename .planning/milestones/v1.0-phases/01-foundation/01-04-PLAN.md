**Schema** — per CONTEXT.md locked decisions (full coverage of all API attributes):

**Per-resource timeouts block (CONTEXT.md locked decision):**
Add `timeouts.Block(ctx, timeouts.Opts{Create: true, Read: true, Update: true, Delete: true})` to the schema attributes. This uses `github.com/hashicorp/terraform-plugin-framework-timeouts/resource/timeouts`. Import `timeouts` from that package. This enables users to configure per-operation timeouts:
```hcl
resource "flashblade_file_system" "example" {
  name        = "test"
  provisioned = 1073741824
  timeouts {
    create = "30m"
    read   = "5m"
    update = "30m"
    delete = "60m"
  }
}
```

- `id` — String, Computed, UseStateForUnknown plan modifier (stable after creation).
- `name` — String, Required. User-facing key. No RequiresReplace (in-place rename supported).
- `provisioned` — Int64, Required. Provisioned size in bytes.
- `destroyed` — Bool, Computed. Shows if file system is soft-deleted outside Terraform.
- `destroy_eradicate_on_delete` — Bool, Optional, Default true. Controls Delete behavior.
- `time_remaining` — Int64, Computed. Milliseconds until auto-eradication (nil if not destroyed).
- `created` — Int64, Computed, UseStateForUnknown.
- `nfs` — SingleNestedAttribute, Optional, Computed:
  - `enabled` — Bool, Optional, Default true.
  - `v3_enabled` — Bool, Optional, Default false.
  - `v4_1_enabled` — Bool, Optional, Default true.
  - `rules` — String, Optional, Default "*(rw,no_root_squash)".
  - `transport` — String, Optional, Default "tcp".
- `smb` — SingleNestedAttribute, Optional, Computed:
  - `enabled` — Bool, Optional, Default false.
  - `access_based_enumeration_enabled` — Bool, Optional, Default false.
  - `continuous_availability_enabled` — Bool, Optional, Default false.
  - `smb_encryption_enabled` — Bool, Optional, Default false.
- `http` — SingleNestedAttribute, Computed (read-only, API-managed):
  - `enabled` — Bool, Computed.
- `space` — SingleNestedAttribute, Computed (read-only):
  - `data_reduction` — Float64, Computed.
  - `snapshots` — Int64, Computed.
  - `total_physical` — Int64, Computed.
  - `unique` — Int64, Computed.
  - `virtual` — Int64, Computed.
  - `snapshots_effective` — Int64, Computed.
- `multi_protocol` — SingleNestedAttribute, Optional, Computed:
  - `access_control_style` — String, Optional.
  - `safeguard_acls` — Bool, Optional.
- `default_quotas` — SingleNestedAttribute, Optional, Computed:
  - `group_quota` — Int64, Optional.
  - `user_quota` — Int64, Optional.
- `nfs_export_policy` — String, Optional. Policy name (plain string per CONTEXT.md).
- `smb_share_policy` — String, Optional. Policy name.
- `promotion_status` — String, Computed.
- `source` — SingleNestedAttribute, Computed.
- `writable` — Bool, Computed.

Use `resource.UseStateForUnknown()` plan modifier on all Computed-only fields (id, created, space.*, http.*, promotion_status, writable, source).

Use explicit defaults in schema matching API defaults per CONTEXT.md decision.
